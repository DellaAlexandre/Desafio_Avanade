# Desafio_Avanade

## Projeto Professor Curso 

O Projeto tem como Finalidade criar um CRUD via API conectado com o Banco de dados: 

- GET (Buscar)
- POST (Criar)
- DELETE (Deletar)
- PUT (Atualizar)

## Regra de Negócio 

> Não é possível cadastrar um Professor em mais de um curso !
